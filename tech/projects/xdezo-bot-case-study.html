Project Overview

The Xdezo Bot is an automated digital concierge for Xdezo Technology, based in Pokhara. It was engineered to bridge the gap between students and the institution, providing 24/7 access to course details, pricing, and enrollment without requiring physical presence or staff intervention.

The Challenge

Educational institutions often struggle with manual enrollment queues and repetitive queries. The challenge was to create a bot that could handle complex, multi-step user data collection while ensuring the data survived a bot restart or server migration.

  • Asynchronous State Management: Collecting student details (Name, Email, Phone, Course) in a specific sequence without losing context.
  • Instant Validation: Creating a secure method for students to verify the authenticity of their course certificates.
  • Scalable Deployment: Hosting the bot in a live environment capable of handling simultaneous user interactions.

Technical Execution

I utilized the pyTelegramBotAPI to build the command architecture and integrated mysql-connector to funnel student data directly into a production-grade MySQL database on PythonAnywhere.

Deployment Strategy

The bot is deployed on PythonAnywhere, utilizing a robust while True polling loop with error handling to ensure maximum uptime and immediate response to user callback queries.

The Solution

  • Automated Enrollment: Students can enroll in any of the 15+ courses (Full Stack, UI/UX, Mobile Dev) directly through the Telegram interface.
  • Security & Verification: Implemented a Certificate Verification system allowing users to validate credentials using unique IDs.
  • Dynamic Information Hub: Integrated an InlineKeyboardMarkup menu that provides instant info on pricing discounts (up to 40% off), location, and internship eligibility.
