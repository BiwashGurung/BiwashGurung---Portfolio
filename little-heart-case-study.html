Project Overview

Little Heart Pet Shop is a premier pet grooming service in Pokhara. The goal was to replace a manual phone-based booking system with a robust automated platform to handle high demand.

The Challenge

The client faced issues with double-booking and "dead hours" where the staff was free but no customers were scheduled. They needed a system that could manage specific time slots for dog grooming sessions.

The Solution

  • Backend: Built a custom appointment logic in Django to prevent scheduling conflicts.
  • Database: Relational MySQL structure to track customer history and pet details.
  • Frontend: Mobile-first UI using Bootstrap to ensure customers can book while on the go.

Technical Execution

One of the key technical hurdles was handling the media content—filming and editing sessions to be displayed on the site. I integrated a media pipeline that showcased recent grooming results to build trust with new users.
